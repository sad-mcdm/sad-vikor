Elicitação Precisa

{% if config.SYSTEM_MODE == 'central' %}

Apoio a múltiplos formatos de atração: pesos diretos, ranqueamento ordinal (ROC), comparação qualitativa MACBETH, e método da bisseção para funções de valor no BWT.

{% elif config.SYSTEM_MODE == 'smarts_smarter' %}

Defina o peso dos seus critérios informando notas diretas (0 a 100) baseadas em Swings ou simplesmente ordene a importância para obter pesos ótimos via ROC.

{% elif config.SYSTEM_MODE == 'ahp' %}

Compare os critérios par a par usando a escala padrão de Saaty de 1 a 9, gerando o autovetor principal pelo método da média geométrica normalizada.

{% elif config.SYSTEM_MODE == 'bwm' %}

Identifique o melhor e o pior critério, e faça apenas 2n-3 comparações par a par, otimizando o esforço cognitivo do decisor.

{% elif config.SYSTEM_MODE == 'macbeth' %}

Elicite atratividade usando categorias semânticas (nula, muito fraca, fraca, moderada, forte, muito forte, extrema) resolvidas matematicamente via PL.

{% elif config.SYSTEM_MODE == 'bwt' %}

Trabalhe com trocas compensatórias exatas (tradeoffs) entre o melhor critério e os demais, estimando as curvas de utilidade via bisseções sucessivas.

{% endif %}
